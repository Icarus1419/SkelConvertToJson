# SkelConvertToJson

一个将 Spine `.atlas` 文件和 2D `.skel` 二进制文件转为 `.json` 文件的工具类.

## 支持的版本
- 4.2.33  :green_circle:
- 其他 :warning:

## 如何使用

- 打开"SimpleConsoleProgram.exe"并按照控制台内提示的步骤进行操作.
